Maths

The position relative to the \(x\) axis is defined as: $$x = v \cos(\theta) t$$ The position relative to the \(y\) axis is defined as: $$y = v \sin(\theta) t - \frac{g t^{2}}{2}$$ As \(\Delta y = 0\), the Time of flight can be found by setting the equation for \(y = 0\), $$\Sigma t = \frac{2*v \sin(\theta)}{g}$$ The range (net \(x\) displacement) can be found by substituting \(\Sigma t\): $$x_{max} = v \cos(\theta) \Sigma t $$ \(y_{max}\) occurs when \(v_{y}=0\) therefore: $$v_{y} = \frac{\partial}{\partial t} \left(v \sin(\theta) t - \frac{g t^{2}}{2} \right) \Rightarrow $$ $$v_{y} = v \sin(\theta) - g t = 0 \Rightarrow$$ $$t_{y_{max}} = \frac{v\sin(\theta)}{g}$$ Substituting this value of \(t\) into \(y \Rightarrow\) $$y_{max} = \frac{v^2 \sin^2(\theta)}{2g}$$
